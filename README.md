## 4. Пример кода
```javascript
const n = arr.length;
for (let i = 0; i < n - 1; i++) {
    let minIndex = i;
    for (let j = i + 1; j < n; j++) {
        if (arr[j] < arr[minIndex]) {
            minIndex = j;
        }
    }
    // Меняем местами arr[i] и arr[minIndex]
    [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
}
return arr;
```
